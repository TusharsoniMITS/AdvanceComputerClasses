# Advanced Computer Classes

A full-stack MERN web application designed for typing practice, student management, and course handling.

---

## 🚀 Features

- User Authentication
- Typing Practice System
- Student Dashboard
- Course Management
- Responsive UI
- REST API Integration

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- CSS
- Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB

---

## 📂 Project Structure

client/ → Frontend  
backend/ → Backend APIs

---

## ⚙️ Installation

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
